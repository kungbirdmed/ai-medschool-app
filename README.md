## 🛠️ Install & Dev Setup

```bash
# Clone the repo
git clone https://github.com/kungbirdmed/ai-medschool-app.git
cd ai-medschool-app

# Recommended: Node.js v18+
# Install dependencies (once React app is initialized)
npm install

# Start dev server (if using Vite or Create React App)
npm run dev
```
